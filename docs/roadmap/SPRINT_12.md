Les attributs `data-item-name` et `data-item-eq-name` (utilises par le JS de recherche) conservent volontairement `item.name` non traduit pour preserver la semantique de cle. Tests `ItemLocalizationExtensionTest` (5 cas : traduction matchee, fallback base name, RequestStack vide, Item null, enregistrement du filter).
